PODVIZ · MANGA LAYER

motion-comic prototype — word-synced speech bubbles over a manga panel
0.0 / 0.0s

Placeholder host art is drawn procedurally — drop a real manga panel per beat via beat.src in data/episode.json and it loads instead. Bubble style + reveal mode are live-toggleable, and each beat carries its own default (so the look changes shot-to-shot). Drop an episode.mp3 via the Audio picker to drive timing from real audio. Data model mirrors PodViz (words.json + storyboard beats), so this ports straight into the ffmpeg/ASS renderer.
